# Aruviah - E-commerce Platform

A full-featured e-commerce platform built with React and Firebase.

## Features

- User authentication (Email, Google, Phone)
- Product browsing and search
- Shopping cart and wishlist
- Checkout process
- Order management
- Admin dashboard
- Vendor portal
- Image upload with Cloudinary

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file based on `.env.example` and add your credentials

3. Start the development server:
```bash
npm start
```

## Technologies Used

- React
- Firebase (Auth, Firestore)
- Cloudinary (Image hosting)
- React Router
- Redux Toolkit
- Tailwind CSS
